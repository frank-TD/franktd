# 🔄 选题去重功能说明

## 问题描述

**用户反馈：**
> 今日的热点和我昨天测试生成的内容重复度很高，不符合去重要求

**根本原因：**
- Agent使用的是短期记忆（滑动窗口），每次重启服务或刷新页面就会丢失
- 没有持久化的历史记录存储
- 无法跨会话进行选题去重

---

## ✅ 解决方案

### 新增功能：持久化选题历史记录

**核心组件：**
1. ✅ `src/tools/topic_history.py` - 选题历史管理工具
2. ✅ 更新系统提示词 - 添加强制去重流程
3. ✅ 更新Agent工具列表 - 添加3个新工具

## 🛠️ 新增工具

### 1. `get_recent_topics` - 获取历史选题

**功能：** 查询最近N天内已生成的选题列表

**参数：**
- `days`: 查询天数（默认7天）

**返回：** 历史选题列表

**使用示例：**
```
get_recent_topics(7)
```

**返回示例：**
```
最近7天内已生成的选题（共15个）：

1. 广西三月三怎么玩？跟着嗨萌马逛广西 (2026-04-27)
   分类：泛流量热点
2. 插座小孔不能碰 (2026-04-27)
   分类：儿童安全教育
3. 一张图证明你是养宠人 (2026-04-27)
   分类：泛流量热点
...
```

---

### 2. `check_topic_duplicate` - 检查选题重复

**功能：** 检查某个选题是否在最近7天内已生成

**参数：**
- `topic_title`: 要检查的选题标题

**返回：** 检查结果

**使用示例：**
```
check_topic_duplicate("插座小孔不能碰")
```

**返回示例：**
```
⚠️ 检测到重复或相似选题！

当前选题：插座小孔不能碰

相似的历史选题：
  • 插座小孔不能碰 (2026-04-27) [相似度：高]
```

---

### 3. `record_generated_topic` - 记录新选题

**功能：** 将新生成的选题记录到历史记录中

**参数：**
- `title`: 选题标题
- `category`: 选题分类（泛流量热点/儿童安全教育）
- `platform`: 来源平台

**返回：** 记录结果

**使用示例：**
```
record_generated_topic(
    title="过马路要走斑马线",
    category="儿童安全教育",
    platform="抖音热点宝"
)
```

**返回示例：**
```
✅ 已记录选题：过马路要走斑马线
```

## 📋 去重流程

### 强制去重步骤（Agent自动执行）

```
1. 调用 get_recent_topics(7)
   └─ 获取最近7天已生成的选题列表

2. 搜索热点话题
   └─ 调用 search_all_platforms_hot_topics

3. 对每个候选选题
   ├─ 调用 check_topic_duplicate("选题标题")
   ├─ 如果重复 → 跳过，更换选题
   └─ 如果不重复 → 保留

4. 生成最终选题后
   └─ 对每个新选题调用 record_generated_topic

5. 返回去重后的选题清单
```

## 🗂️ 历史记录存储

### 存储位置

```
workspace/projects/
└── assets/
    └── topic_history.json    # 历史记录文件
```

### 文件格式

```json
{
  "topics": [
    {
      "title": "选题标题",
      "category": "泛流量热点",
      "platform": "抖音热点宝",
      "date": "2026-04-27T10:00:00",
      "id": "topic_20260427100000"
    }
  ],
  "last_updated": "2026-04-27T10:00:00"
}
```

### 数据管理

- ✅ **自动清理：** 只保留最近30天的记录
- ✅ **相似度检测：** 使用Jaccard相似度算法
- ✅ **持久化存储：** JSON文件，重启不丢失

## 🎯 去重规则

### 重复判定标准

1. **完全重复：** 标题完全相同
2. **包含关系：** 一个标题包含另一个标题
3. **相似度高：** Jaccard相似度 > 0.5

### 安全领域轮换（专题B）

- ✅ **近7天内同一安全领域不能重复**
- ✅ 必须轮换不同领域：居家/出行/校园/防拐/防火防电/防溺水/网络/应急/食品/户外
- ✅ 建议轮换顺序：周一居家+出行、周二校园+防拐、周三防火防电+食品安全...

### 热点去重（专题A）

- ✅ **近7天内同一热点不能重复**
- ✅ 即使角度不同，同一热点也视为重复
- ✅ 优先选择当日最新热点

## 💡 常见问题

### Q1: 历史记录文件在哪里？

A: `workspace/projects/assets/topic_history.json`

### Q2: 历史记录会保留多久？

A: 自动保留最近30天的记录，超过30天的会自动清理。

### Q3: 可以手动添加历史记录吗？

A: 可以，直接编辑 `topic_history.json` 文件，按格式添加即可。

### Q4: 如果误删了历史记录怎么办？

A: 可以恢复备份文件，或重新生成选题（会创建新的历史记录）。

### Q5: 相似度算法是怎样的？

A: 使用Jaccard相似度，计算两个标题中共同字符的比例，超过0.5视为相似。
